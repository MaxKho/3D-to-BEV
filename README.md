Run the three scripts consecutively (planes.py --> vps.py --> bev.py), saving all generated files to outputs. You can run all the scripts in one command as follows:

```python planes.py --image "data/Photo.png" --out_dir outputs --cpu; python vps.py --image "data/Photo.png" --out_dir outputs; python bev.py --image "data/Photo.png" --out_dir outputs```
